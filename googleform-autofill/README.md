## Apaan Nih?

Jadi gini, kadang kan kita disuruh ngisi kuesioner atau form yang banyak banget. Nah, script ini bakal bantuin kamu buat ngisi form itu secara **otomatis** dan **cepet**.

Datanya juga ga asal-asalan banget kok. Namanya ngambil dari dataset nama orang Indonesia asli, terus tinggi sama berat badannya juga diitung biar proporsional (ga kurus banget, ga gemuk banget).

## Fitur-Fiturnya

*   **Nama Indo Asli**: Scriptnya download database nama dari Kaggle, jadi namanya kayak "Budi", "Siti", "Agus", bukan "XyZ123".
*   **Logika Pinter Dikit**: Kalau tingginya segini, beratnya disesuain biar BMI-nya normal.
*   **Bisa Borongan**: Mau ngisi 10x? 100x? 300x? Tinggal ketik angkanya, tinggal tidur.
*   **Anti Ribet**: Cuma satu file `main.py`, ga pake folder-folderan yang bikin pusing.

## Cara Pakenya

Gampang banget elah.

1.  **Install Python** dulu kalo belum punya.
2.  Install library yang dibutuhin:
    ```bash
    pip install requests kagglehub
    ```
3.  Jalanin scriptnya:
    ```bash
    python main.py
    ```
4.  Nanti dia bakal nanya mau ngisi berapa kali. Masukin aja angkanya.
5.  Tungguin sampe kelar.
